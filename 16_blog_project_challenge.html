My Journey into Programming

As a cohort 6 member of eHA Academy, I began my journey to become a programmer and software developer. The experience has been incredibly enlightening, providing me with the skills and knowledge needed to thrive in the digital world. It's been a blend of theory, practical projects, and hands-on learning that has prepared me for a career in software development.

Learning Web Development

During my time at eHA Academy, I learned the basics of web development. This included setting up Git and GitHub accounts to manage version control, writing basic HTML, and using CSS to style web pages. I've also gained exposure to web design tools like Figma and the importance of user experience in web development.

Exploring Figma and UI/UX Design

Figma has become an essential tool in my web design process. It allows me to create sleek, functional prototypes while keeping user experience at the forefront. Understanding UI/UX design principles is critical when building applications that are both aesthetically pleasing and easy to navigate for users. This part of my learning journey has shown me how design is an integral part of software development.

Setting Up Git and GitHub

One of the most valuable tools I've learned to use is Git, along with GitHub for version control. These tools allow me to efficiently manage my projects, track changes, and collaborate with other developers. Understanding how to work with Git has made it much easier to handle various aspects of coding, including branching and merging different versions of my work.

Mastering HTML and CSS

Learning HTML and CSS has been an enjoyable experience. HTML forms the backbone of any website, providing structure and meaning to the content. CSS complements it by adding styles, fonts, colors, and layouts. Through eHA Academy, I've been able to experiment with these technologies to build responsive and visually appealing web pages. I now understand how to create efficient code that delivers both functionality and design.
